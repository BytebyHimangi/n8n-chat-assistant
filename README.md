# n8n AI Chat Assistant

## Overview

This project is an AI-powered chatbot workflow built using **n8n** and **Google Gemini API**. It automates user interactions through a visual workflow and generates intelligent responses using Google's Generative AI.

## Features

- AI-powered conversational chatbot
- Google Gemini API integration
- Visual workflow automation using n8n
- Prompt-based response generation
- Easy to customize and extend

## Tech Stack

- n8n
- Google Gemini API
- HTTP Request Node
- JSON Workflow

## Workflow

1. User sends a message.
2. n8n receives the request.
3. The prompt is sent to Google Gemini API.
4. Gemini generates an AI response.
5. The response is returned to the user.
