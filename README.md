# n8n AI Chat Assistant

## Overview
This project is an AI-powered chatbot workflow built using n8n. It automates user interactions by connecting AI services through a visual workflow.

## Features
- AI-powered conversational workflow
- Visual automation using n8n
- API integration
- Easy to customize
- No-code/low-code automation

## Tech Stack
- n8n
- Google Gemini API 
- HTTP Request Node
- Webhook 

## Workflow
1. User sends a message.
2. n8n receives the request.
3. AI processes the prompt.
4. Response is returned to the user.

## Learning Outcomes
- Workflow automation
- AI integration
- Prompt handling
- API integration
